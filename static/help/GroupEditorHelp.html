# The groups are used to group catagories together. For example you have to pay out to several utility companies every month and a group named "Utility" can be prefixed to a catagory entry for each utility. So phone, cellular, water and electricity expenses can be prefixed with "Utility" and all utilities will be grouped together in reports. By creating groups records you can standardize the group names and only have to enter them one time. ---- <p align="center"> ![Groups Screen](./images/groupscreen.png) </p> # Data Fields: <p align="center"> ![Fields](./images/groupfields.png) </p> * #### Primary Key - the numeric record number unique to a record. * #### Description - the group name like "Utility" or "Insurance" # Make a new entry: * #### 1.
