# Catagories make possible for you to group your expenses fo reporting. For instance a catagory name "Car Insurance" can be applied to all expenses for car insurance and you can ask the report generator for a list of all car insurance for a given time period. Also see the groups feature which all further grouping by combining a group with a catagory. ---- <p align="center"> ![Catagories Screen](./images/catagoriesscreen.png) </p> #Data Fields <p align="center"> ![Fields](./images/catagoryfields.png) </p> * #### Primary Key - the numeric record number unique to a record. * #### Description - about the group i.e. Phone Bill * #### Group - select from this list of groups and it wil be prefixed to the description when you insert a new record. # Make a new entry: * #### 1. Enter a Description or just select a group i.e. Utilities to group all utility expenses together. * #### 2. Optionally select a group as prefix to the description. * #### 3. Click the Insert button.
